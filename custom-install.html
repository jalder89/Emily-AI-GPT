Custom Installation

Setup Instructions
  1. Clone the repo with git clone https://github.com/jalder89/Slack-Express-App-OpenAI.git
  2. Install dependencies with npm install
  3. Create a Slack App at https://api.slack.com/apps
  4. Add chat:write and history Bot Scopes. Add the history scopes for channel types you want Emily to watch.
  5. Enable Events
  6. Setup forwarding with ngrok http 3000
  7. Add your forwarding host address as the Request URL
  8. Subscribe to desired message events
  9. Install the app to your Slack workspace
  10. Add the Bot User OAuth Access Token and Signing Secret to a .env file in the app code
  11. Run the app with npm start
  12. Send a message to the bot in Slack
